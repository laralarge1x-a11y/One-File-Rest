## 🚀 REPLIT DEPLOYMENT (7 STEPS)

### 1. Push to GitHub
```bash
git init
git add .
git commit -m "Elite Tok Club"
git push origin main
```

### 2. Import to Replit
- replit.com → Import from GitHub
- Paste repo URL → Import

### 3. Add Secrets
- Click "Secrets"
- Add all environment variables

### 4. Install
```bash
npm install
cd client && npm install && cd ..
```

### 5. Build
```bash
cd client && npm run build && cd ..
```

### 6. Migrate
```bash
npm run migrate
```

### 7. Deploy
```bash
npm start
```
